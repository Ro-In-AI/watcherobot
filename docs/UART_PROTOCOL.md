# WatcherRobot UART 通讯协议设计

## 1. 概述

本协议定义了上位机（PC/MCU）与 ESP32 下位机之间的 UART 通讯规范，用于控制舵机、播放预设动作、查询状态等。

### 1.1 协议特性

| 特性 | 说明 |
|------|------|
| 通讯方式 | UART (TTL 3.3V) |
| 波特率 | 115200 (默认), 921600 (可选) |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | 无 |
| 流控制 | 无 |
| 字节序 | 大端 (MSB First) |

### 1.2 协议模式

支持两种协议模式：

| 模式 | 用途 | 格式 |
|------|------|------|
| **文本模式** | 调试/日志输出 | ASCII 字符串 |
| **二进制模式** | 高效数据传输 | 二进制帧 |

---

## 2. 二进制帧格式

### 2.1 帧结构

```
┌──────────┬──────────┬──────────┬────────────┬──────────────┬──────────┐
│  帧头    │  长度    │  功能码   │   载荷      │    CRC16     │  帧尾    │
│  2字节   │  1字节   │  1字节    │  N字节      │    2字节     │  1字节   │
│  0xAA55  │  LEN     │  CMD     │   DATA     │    CRC       │   0x0D   │
└──────────┴──────────┴──────────┴────────────┴──────────────┴──────────┘
```

### 2.2 字段说明

| 字段 | 字节数 | 说明 |
|------|--------|------|
| 帧头 | 2 | 固定 `0xAA 0x55` |
| 长度 | 1 | 载荷长度 (0-255) |
| 功能码 | 1 | 命令/响应类型 |
| 载荷 | N | 数据内容 |
| CRC16 | 2 | Modbus CRC16 (CRC-16/MODBUS) |
| 帧尾 | 1 | 固定 `0x0D` (CR) |

### 2.3 示例帧

**设置舵机角度 (CMD=0x01)**
```
AA 55 06 01 00 5A 01 F4 4E 0D
│  │  │  │  │  │  │  │  └─ 帧尾
│  │  │  │  │  │  │  └──── CRC16 低字节
│  │  │  │  │  │  └──────── CRC16 高字节
│  │  │  │  │  └────────── 持续时间 LSB (500ms)
│  │  │  │  └───────────── 角度值 (90°)
│  │  │  └──────────────── 舵机ID (0)
│  │  └─────────────────── 载荷长度 6
│  └─────────────────────── 功能码
└────────────────────────── 帧头
```

## 3. 功能码定义

### 3.1 命令码 (上位机 → 下位机)

| 功能码 | 命令名称 | 载荷格式 | 说明 |
|--------|----------|----------|------|
| 0x01 | `SET_SERVO` | `<ID:u8><Angle:u8><Duration:u16>` | 设置单路舵机角度 |
| 0x02 | `SET_SERVO_ALL` | `<ID0:u8><Angle0:u8>...<IDn:u8><AngleN:u8>` | 设置多路舵机 |
| 0x03 | `SERVO_MOVE` | `<ID:u8><Direction:i8>` | 连续移动 (1/-1/0) |
| 0x04 | `SERVO_STOP` | `<ID:u8>` | 停止指定舵机 |
| 0x05 | `SERVO_STOP_ALL` | - | 停止所有舵机 |
| 0x10 | `PLAY_ACTION` | `<ActionID:u8>` | 播放预设动作 |
| 0x11 | `STOP_ACTION` | - | 停止当前动作 |
| 0x12 | `QUEUE_ADD` | `<ID:u8><Angle:u8><Duration:u16>` | 添加到动作队列 |
| 0x13 | `QUEUE_CLEAR` | - | 清空动作队列 |
| 0x20 | `GET_STATUS` | - | 查询设备状态 |
| 0x21 | `GET_SERVO_POS` | `<ID:u8>` | 查询舵机当前位置 |
| 0x22 | `GET_VERSION` | - | 获取固件版本 |
| 0x30 | `SET_BUZZER` | `<Frequency:u16><Duration:u16>` | 蜂鸣器控制 |
| 0x31 | `SET_LED` | `<LED_ID:u8><R:u8><G:u8><B:u8>` | LED 控制 |
| 0x40 | `SAVE_NVS` | - | 保存配置到 NVS |
| 0x41 | `RESET_NVS` | - | 重置 NVS 配置 |
| 0xF0 | `REBOOT` | - | 重启设备 |

### 3.2 响应码 (下位机 → 上位机)

| 功能码 | 响应名称 | 载荷格式 | 说明 |
|--------|----------|----------|------|
| 0x81 | `ACK` | `<CMD:u8><Status:u8>` | 命令确认 |
| 0x82 | `ERROR` | `<CMD:u8><ErrorCode:u8>` | 命令错误 |
| 0x90 | `STATUS_RESP` | `<ServoCount:u8><ActionPlaying:u8><QueueSize:u8>` | 设备状态响应 |
| 0x91 | `SERVO_POS_RESP` | `<ID:u8><Angle:u8>` | 舵机位置响应 |
| 0x92 | `VERSION_RESP` | `<Major:u8><Minor:u8><Patch:u8>` | 版本响应 |
| 0xA0 | `EVENT_ACTION_DONE` | `<ActionID:u8>` | 动作播放完成事件 |
| 0xA1 | `EVENT_SERVO_REACHED` | `<ID:u8><Angle:u8>` | 舵机到达目标位置 |

## 4. 数据类型定义

### 4.1 基本类型

| 类型 | 字节数 | 范围 | 说明 |
|------|--------|------|------|
| `u8` | 1 | 0 - 255 | 无符号 8 位整数 |
| `i8` | 1 | -128 - 127 | 有符号 8 位整数 |
| `u16` | 2 | 0 - 65535 | 无符号 16 位整数 (大端) |
| `i16` | 2 | -32768 - 32767 | 有符号 16 位整数 (大端) |

## 5. 详细命令格式

### 5.1 SET_SERVO - 设置单路舵机

**请求帧:**
```
AA 55 06 01 [ID:u8][Angle:u8][Duration:u16] [CRC16] 0D
```

| 参数 | 类型 | 说明 |
|------|------|------|
| ID | u8 | 舵机 ID (0: X轴, 1: Y轴) |
| Angle | u8 | 角度值 (0-180°) |
| Duration | u16 | 持续时间 (ms), 0 表示立即 |

**响应帧 (ACK):**
```
AA 55 03 81 [CMD:u8][Status:u8] [CRC16] 0D
```

| Status 值 | 说明 |
|-----------|------|
| 0x00 | 成功 |
| 0x01 | 无效舵机 ID |
| 0x02 | 无效角度值 |
| 0x03 | 舵机忙碌 |

### 5.2 SERVO_MOVE - 连续移动

**请求帧:**
```
AA 55 03 03 [ID:u8][Direction:i8] [CRC16] 0D
```

| 参数 | 类型 | 说明 |
|------|------|------|
| ID | u8 | 舵机 ID |
| Direction | i8 | 方向: 1=正向, -1=反向, 0=停止 |

### 5.3 PLAY_ACTION - 播放预设动作

**请求帧:**
```
AA 55 02 10 [ActionID:u8] [CRC16] 0D
```

| ActionID | 预设动作 |
|----------|----------|
| 0x00 | 复位/归位 |
| 0x01 | 打招呼 |
| 0x02 | 摇头 |
| 0x03 | 点头 |
| 0x04 | 环顾四周 |
| 0x05-0xFF | 自定义动作 |

### 5.4 GET_STATUS - 查询状态

**请求帧:**
```
AA 55 01 20 [00] [CRC16] 0D
```

**响应帧:**
```
AA 55 04 90 [ServoCount:u8][ActionPlaying:u8][QueueSize:u8] [CRC16] 0D
```

| 字段 | 说明 |
|------|------|
| ServoCount | 舵机数量 |
| ActionPlaying | 当前是否有动作在播放 (0/1) |
| QueueSize | 动作队列剩余容量 |

## 6. CRC 校验

### 6.1 CRC16 Modbus

```c
uint16_t crc16_modbus(const uint8_t *data, uint16_t len) {
    uint16_t crc = 0xFFFF;
    for (uint16_t i = 0; i < len; i++) {
        crc ^= data[i];
        for (uint8_t j = 0; j < 8; j++) {
            if (crc & 0x0001) {
                crc = (crc >> 1) ^ 0xA001;
            } else {
                crc >>= 1;
            }
        }
    }
    return crc;
}
```

### 6.2 CRC 计算范围

CRC 计算范围: `长度(1) + 功能码(1) + 载荷(N)`

**示例:**
```
帧: AA 55 06 01 00 5A 01 F4 [CRC] 0D
     └─────── CRC 计算范围 ───────┘
```

## 7. 文本模式 (调试用)

### 7.1 格式

```
$CMD:PARAM1:PARAM2:PARAM3\r\n
```

### 7.2 可用命令

| 命令 | 示例 | 说明 |
|------|------|------|
| SET_SERVO | `$SET_SERVO:0:90:500` | 设置舵机 |
| SERVO_MOVE | `$SERVO_MOVE:0:1` | 连续移动 |
| PLAY_ACTION | `$PLAY_ACTION:1` | 播放动作 |
| QUEUE_ADD | `$QUEUE_ADD:0:90:500` | 添加队列 |
| GET_STATUS | `$GET_STATUS` | 获取状态 |
| GET_VERSION | `$GET_VERSION` | 获取版本 |

### 7.3 响应格式

```
+OK:CMD:PARAM1:PARAM2
-ERR:ERROR_CODE:DESCRIPTION
```

|--------|------|

## 9. 通讯流程示例

### 9.1 设置舵机

```
上位机 → 下位机: AA 55 06 01 00 5A 01 F4 4E 0D
         (设置舵机0到90°, 持续500ms)

下位机 → 上位机: AA 55 03 81 01 00 5A 0D
         (ACK 成功)
```

### 9.2 查询状态

```
上位机 → 下位机: AA 55 01 20 00 24 0D

下位机 → 上位机: AA 55 04 90 02 00 00 5A 0D
         (2个舵机, 无动作在播放, 队列空)
```

### 9.3 动作完成事件

```
下位机 → 上位机: AA 55 02 A0 01 5A 0D
         (动作1播放完成)
```
