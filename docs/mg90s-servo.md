# MG90S 舵机技术文档

## 1. 规格参数

| 参数 | 数值 |
|------|------|
| 控制频率 | 50Hz (20ms周期) |
| 脉宽范围 | 1ms-2ms |
| 角度范围 | 0°-180° |
| 供电电压 | 4.8V-6V |
| 最大速度 | ~0.11s/60° (在4.8V下) |

## 2. 速度控制方式

### 2.1 软件渐变（推荐）

通过分多次逐步改变 PWM 占空比，在指定时间内到达目标角度：

```
总时间 = 移动时间 (ms)
步数 = 总时间 / 步进间隔 (通常 20ms)
每步角度增量 = (目标角度 - 当前位置) / 步数
```

**优点**：
- 精确控制移动时间
- 运动平滑

**缺点**：
- 占用 CPU 资源
- 最小步长受死区限制

### 2.2 硬件局限

#### 最小稳定步长

MG90S 这种小型舵机的齿轮组有一定 **死区 (Deadband)**：
- 如果步进太小（比如 <0.5°），舵机可能不会响应
- 需要误差累积到一定程度才会"抖动"一下

#### 最大速度限制

舵机的最高速度由电机电压和减速比物理决定：
- **MG90S 在 4.8V 下速度约为 0.11s/60°**
- 无法通过软件让它快过这个物理极限

### 2.3 降低供电电压

降低供电电压可以减慢速度：
- 但会降低扭矩
- 可能导致舵机运行不稳定
- 不推荐

## 3. 建议

1. **最小移动时间**：建议 >= 100ms，避免死区问题
2. **平滑移动步长**：每步 >= 1°，确保舵机响应
3. **最大速度**：约 500°/秒（计算：60°/0.11s ≈ 545°/s）
4. **平滑算法**：
   - 时间范围：100ms - 2000ms
   - 步进间隔：20ms（一个 PWM 周期）
   - 最大步数：100 步

## 4. ESP32 实现参考

```c
// 伪代码示例
void servo_smooth_move(uint8_t target_angle, uint16_t duration_ms) {
    uint8_t current = get_current_angle();
    int16_t diff = target_angle - current;
    uint16_t steps = duration_ms / 20;  // 20ms 步进
    if (steps < 1) steps = 1;
    int16_t step_size = diff / steps;

    for (int i = 0; i < steps; i++) {
        current += step_size;
        set_pwm_duty(current);
        vTaskDelay(20 / portTICK_PERIOD_MS);
    }
    set_pwm_duty(target_angle);  // 确保到达精确目标
}
```
